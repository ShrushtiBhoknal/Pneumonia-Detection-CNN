# Pneumonia Detection using CNN
This project builds a Convolutional Neural Network (CNN) model to detect Pneumonia from Chest X-ray images.

## Overview
- Binary classification: Normal vs Pneumonia
- Transfer Learning using MobileNetV2
- Model evaluation using Confusion Matrix and Accuracy

## Techniques Used
- Image preprocessing
- Data augmentation
- Transfer learning
- Model evaluation

## Technologies
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Seaborn
